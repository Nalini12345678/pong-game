# Pong Game Documentation

## Introduction
Pong is one of the earliest arcade video games, simulating table tennis. Players control paddles to hit a ball back and forth, striving to score points against their opponent.

## How to Play
1. Each player controls a paddle on either side of the screen.
2. The objective is to score points by getting the ball past the opponent's paddle.
3. Players earn a point when the opponent fails to return the ball.
4. Play continues until a predetermined score is reached.

## Features
- Attractive graphics and smooth gameplay
- Customizable controls
- Single-player and multiplayer modes
- Score tracking 

## Controls
- **Player 1:**
    - Move Up: W key
    - Move Down: S key
- **Player 2:**
    - Move Up: Up Arrow key
    - Move Down: Down Arrow key
